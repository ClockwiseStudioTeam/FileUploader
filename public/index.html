File Uploader API

This is a simple API for uploading files and storing them with MongoDB. It's designed to be easily integrated with Webflow.

API Endpoints

  • POST /.netlify/functions/api/upload - Upload a file
  • GET /.netlify/functions/api/files/:uuid - Get file by UUID
  • GET /.netlify/functions/api/files - List all files

Integration with Webflow

To integrate this file uploader with your Webflow site, you have two options:

Option 1: Using the Embed Element

  1. Add an Embed element to your Webflow page
  2. Copy the contents of webflow-upload.html into the Embed element
  3. Update the API_URL variable to point to this deployed API

Option 2: Using Custom Code

  1. Create a div element with ID file-upload-target in your Webflow page
  2. Go to Project Settings > Custom Code
  3. Add the code from webflow-embed-code.js to the "Before </body> tag" section
  4. Update the API_URL variable to point to this deployed API
